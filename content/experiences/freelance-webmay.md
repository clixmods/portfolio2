## Description de la mission

Mission freelance sur Time Trial Cards, un jeu éducatif mobile développé par Webmay où les joueurs trient les grands événements de l'Histoire sur une frise chronologique. J'ai rejoint le projet en cours de production pour renforcer le développement sur des sujets gameplay, UI/UX, data et stabilisation.

### Projet principal

#### Time Trial Cards

- Développement et évolution de fonctionnalités dans une base de code existante, avec intégration pragmatique dans une architecture Unity en production
- Refonte partielle de l'architecture technique : persistance unifiée, migration de sauvegardes vers JSON, refactoring ciblé et optimisations de performance
- Mise en place et évolution du pipeline de données via Google Sheets, avec validation des données, parsing de nouveaux champs et prise en charge des filtres métier
- Développement d'outils internes pour faciliter la validation, le debug et le suivi qualité, dont un système de report de bugs in-game et des outils de contrôle des données et des niveaux
- Travail sur la localisation FR/EN, les tutoriels, le mode Entraînement, l'économie du jeu, ainsi que la correction de bugs bloquants et l'amélioration de la robustesse globale sur mobile
- Gestion des builds Android et du déploiement de versions de test via Google Play Console, avec suivi du versioning et préparation du release flow

### Compétences développées

- Développement gameplay et intégration dans une base de code en production
- Architecture technique et refactoring ciblé sur Unity
- Pipeline de données et outillage interne
- Optimisation des performances pour mobile
- Déploiement Android et gestion du release flow
