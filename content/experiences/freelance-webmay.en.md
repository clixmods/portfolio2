## Mission Description

Freelance mission on Time Trial Cards, an educational mobile game developed by Webmay where players sort major historical events on a timeline. I joined the project mid-production to strengthen development on gameplay, UI/UX, data, and stabilization topics.

### Main Project

#### Time Trial Cards

- Development and evolution of features within an existing codebase, with pragmatic integration into a Unity architecture in production
- Partial refactoring of the technical architecture: unified persistence, save migration to JSON, targeted refactoring and performance optimizations
- Setup and evolution of the data pipeline via Google Sheets, with data validation, new field parsing and business filter support
- Development of internal tools to facilitate validation, debugging and quality tracking, including an in-game bug reporting system and data/level control tools
- Work on FR/EN localization, tutorials, Training mode, game economy, as well as blocking bug fixes and overall mobile robustness improvements
- Android build management and test version deployment via Google Play Console, with versioning tracking and release flow preparation

### Skills Developed

- Gameplay development and integration into a production codebase
- Technical architecture and targeted refactoring on Unity
- Data pipeline and internal tooling
- Mobile performance optimization
- Android deployment and release flow management
