# Plateforme de gestion d'événements

## Description du projet

Projet académique en équipe développé à l'IUT Montpellier-Sète. Application web complète permettant la gestion des événements, des inscriptions et du suivi des participants avec une architecture découplée moderne : back-end Symfony/API Platform et front-end Vue.js.

## Développement back-end

### API REST sécurisée
- **Conception et implémentation** d'une API REST complète
- **Authentification JWT** robuste avec gestion des tokens
- **Gestion des rôles utilisateurs** (admin, organisateur, participant)
- **Sécurisation** des endpoints avec contrôle d'accès

### Architecture SOLID
- **Mise en œuvre des principes SOLID** pour un code structuré et maintenable
- **Séparation des responsabilités** claire entre les couches
- **Injection de dépendances** pour une architecture modulaire
- **Tests unitaires** et validation des fonctionnalités

### Endpoints métier
- **Gestion des événements** : création, modification, suppression
- **Système d'inscription** avec gestion des places disponibles
- **Suivi des participants** et statistiques
- **Notifications** automatiques par email

## Développement front-end

### Interface Vue.js moderne
- **Construction d'une SPA** dynamique avec Vue.js
- **Vue Router** pour une navigation fluide
- **Composants réutilisables** et architecture modulaire
- **Gestion d'état** centralisée avec Vuex

### Intégration API
- **Appels API sécurisés** avec gestion des tokens JWT
- **Gestion des erreurs** et feedback utilisateur
- **Interface responsive** adaptée mobile et desktop
- **Validation côté client** synchronisée avec le backend

## Infrastructure

### Conteneurisation Docker
- **Conteneurisation complète** avec Docker & Docker Compose
- **Environnement isolé** et reproductible
- **Déploiement local automatisé** avec gestion des services
- **Orchestration** front/back/database

### Base de données
- **Modélisation relationnelle** optimisée
- **Migrations** structurées et versionées
- **Contraintes d'intégrité** et validation des données
- **Performances** optimisées avec indexation

## Collaboration & gestion de projet

### Travail en équipe
- **Trinôme structuré** avec répartition claire des responsabilités
- **Communication régulière** et synchronisation d'équipe
- **Revues de code** systématiques
- **Pair programming** sur les fonctionnalités complexes

### Outils de développement
- **GitHub** pour le versionnage et collaboration
- **Branches feature** avec workflow Git Flow
- **Issues tracking** et gestion des tâches
- **CI/CD** avec tests automatisés

## Technologies utilisées

- **Symfony 6** : Framework backend PHP
- **API Platform** : Création d'APIs REST/GraphQL
- **Vue.js 3** : Framework frontend réactif
- **Docker** : Conteneurisation et déploiement
- **MySQL** : Base de données relationnelle
- **JWT** : Authentification sécurisée

## Compétences développées

Ce projet m'a permis de maîtriser :
- **Architecture découplée** frontend/backend
- **Authentification moderne** avec JWT
- **Principes SOLID** appliqués en pratique
- **Collaboration** en équipe de développement
- **DevOps** avec Docker et CI/CD

## Résultats

Application complète et fonctionnelle démontrant une maîtrise des technologies web modernes et des bonnes pratiques de développement en équipe.
