# Test Layout System

*Complete demonstration of the new information box system with all sizes and features.*

This test project presents all the features of the layout system developed for the portfolio. It serves as a reference to validate all use cases and box sizes.

## Tested Features

### Size System
- **size-small**: Compact boxes (3 columns)
- **size-medium**: Standard size (4 columns)
- **size-large**: Prominent boxes (6 columns)
- **size-xl**: Full showcase (8 columns)

### Special Previews
- **Contributors**: Overlapping avatars with counter
- **Gallery**: Automatic image carousel
- **Testimonials**: Automatic comment rotation
- **YouTube**: Preview with integrated modal player

### Responsive Design
- **Desktop**: 12-column grid
- **Tablet**: 8-column grid
- **Mobile**: Single column
