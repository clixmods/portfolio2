# Event Management Platform

## Project Description

Team academic project developed at IUT Montpellier-Sète. Complete web application allowing event management, registrations and participant tracking with a modern decoupled architecture: Symfony/API Platform backend and Vue.js frontend.

## Backend Development

### Secure REST API
- **Design and implementation** of a complete REST API
- **Robust JWT authentication** with token management
- **User role management** (admin, organizer, participant)
- **Endpoint security** with access control

### SOLID Architecture
- **Implementation of SOLID principles** for structured and maintainable code
- **Clear separation of responsibilities** between layers
- **Dependency injection** for modular architecture
- **Unit tests** and functionality validation

### Business Endpoints
- **Event management**: creation, modification, deletion
- **Registration system** with available places management
- **Participant tracking** and statistics
- **Automatic notifications** by email

## Frontend Development

### Modern Vue.js Interface
- **Building a dynamic SPA** with Vue.js
- **Vue Router** for fluid navigation
- **Reusable components** and modular architecture
- **Centralized state management** with Vuex

### API Integration
- **Secure API calls** with JWT token management
- **Error handling** and user feedback
- **Responsive interface** adapted for mobile and desktop
- **Client-side validation** synchronized with backend

## Infrastructure

### Docker Containerization
- **Complete containerization** with Docker & Docker Compose
- **Isolated and reproducible environment**
- **Automated local deployment** with service management
- **Front/back/database orchestration**

### Database
- **Optimized relational modeling**
- **Structured and versioned migrations**
- **Integrity constraints** and data validation
- **Optimized performance** with indexing

## Collaboration & Project Management

### Teamwork
- **Structured trio** with clear responsibility distribution
- **Regular communication** and team synchronization
- **Systematic code reviews**
- **Pair programming** on complex features

### Development Tools
- **GitHub** for versioning and collaboration
- **Feature branches** with Git Flow workflow
- **Issues tracking** and task management
- **CI/CD** with automated tests

## Technologies Used

- **Symfony 6**: PHP backend framework
- **API Platform**: REST/GraphQL API creation
- **Vue.js 3**: Reactive frontend framework
- **Docker**: Containerization and deployment
- **MySQL**: Relational database
- **JWT**: Secure authentication

## Skills Developed

This project allowed me to master:
- **Decoupled architecture** frontend/backend
- **Modern authentication** with JWT
- **SOLID principles** applied in practice
- **Collaboration** in development team
- **DevOps** with Docker and CI/CD

## Results

Complete and functional application demonstrating mastery of modern web technologies and good development practices in a team setting.
